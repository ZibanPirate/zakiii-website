# zakiii-website

The code for www.zakiii.com

## Website structure
```
/ : landing page
    - header: (cover pic, name, and resume)
    - nav buttons: (Developer, Reader, Astronaut)
    - articles
    - books
    - contact

/Developer : developer page (resume, portfolio/projects, articles)
    - header: (cover pic, name, and title)
    - resume
    - projects
    - reviews
    - contact
    - linkedIn, fb, YT

    /CV: cv page for employers

/Reader : reader page (my story with books, books)
    - header: (cover pic, name, and resume)
    - why books, why paper books
    - books
    - contact
    - fb, instagram, linkedIn

/Articles : articles page
    - cover pic
    - title
    - description
    - content
    - contact
    - fb, YT, linkedIn, instagram
    - comments

/Projects : projects page
    - cover pic
    - title
    - description
    - what i did
    - contact
    - fb, YT, linkedIn

/Books : books page
    - cover pic
    - title
    - description
    - content
    - contact
    - fb, instagram, YT, linkedIn
    - comments
```
