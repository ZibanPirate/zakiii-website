# zakiii-website

The code for www.zakiii.com

## Website structure
```
/ : landing page
    - header: (cover pic, name, and resume)
    - nav buttons: (Developer, Reader, Astronaut)
    - articles
    - books
    - contact

/Developer : developer page (resume, portfolio/projects, articles)
    - header: (cover pic, name, and title)
    - resume
    - projects
    - reviews
    - contact
    - linkedIn, fb, YT, GH, twitter

    /CV: cv page for employers

/Reader : reader page (my story with books, books)
    - header: (cover pic, name, and resume)
    - why books, why paper books
    - books
    - contact
    - fb, instagram, linkedIn, twitter

/Articles
    - sidebar
    - content (description, or Article)
    /:articleSlug : articles page
        - cover pic
        - title
        - description
        - content
        - contact
        - fb, YT, GH, linkedIn, instagram, twitter
        - comments

/Projects
    - sidebar
    - content (description, or Project)
    /:projectSlug : projects page
        - cover pic
        - title
        - description
        - what i did
        - contact
        - fb, GH, YT, linkedIn, twitter

/Books
    - sidebar
    - content (description, or Book)
    /:bookSlug : books page
        - cover pic
        - title
        - description
        - content
        - contact
        - fb, instagram, YT, linkedIn, twitter
        - comments
```
